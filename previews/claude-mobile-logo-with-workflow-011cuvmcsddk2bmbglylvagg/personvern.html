Personvernerklæring for Tråkke

Sist oppdatert: 22. oktober 2025

Kort oppsummert: Tråkke er designet med personvern i fokus. Vi lagrer kun nødvendige data lokalt på din enhet, og all kommunikasjon med eksterne tjenester skjer med norske og europeiske datakildeer. Vi deler ikke dine data med tredjeparter for markedsføring eller sporing.

1. Behandlingsansvarlig

Tråkke er en åpen kildekode-applikasjon utviklet og driftet av:

  • Prosjekt: Tråkke
  • GitHub: github.com/elzacka/trakke-react
  • Kontakt: Via GitHub Issues eller prosjektets repository

2. Hvilke personopplysninger samler vi inn?

2.1 Data lagret lokalt på din enhet

Tråkke lagrer følgende informasjon i nettleserens lokale lagring (localStorage) på din enhet:

Type data Formål Lagringssted
Kartvisningsinnstillinger Husker karttype, zoom-nivå og siste posisjon localStorage
Brukerpreferanser Språk, enheter (km/mi), og visningsvalg localStorage
Søkehistorikk Raskere tilgang til tidligere søk localStorage
Installasjonsstatus Unngå å vise PWA-installasjonsprompt flere ganger localStorage
Cache-data (PWA) Offline-funksjonalitet og raskere lasting Service Worker Cache

Viktig: Alle disse dataene lagres kun på din enhet og sendes aldri til eksterne servere. Du kan slette alle data når som helst ved å bruke knappen "Slett mine data" i appens innstillinger.

2.2 Posisjonsdata

Hvis du velger å aktivere stedstjenester:

  • Tråkke ber om tillatelse til å bruke GPS/stedstjenester via nettleserens Geolocation API
  • Din nåværende posisjon brukes kun til å sentrere kartet og vise din plassering
  • Posisjonsdata lagres aldri permanent og sendes aldri til eksterne servere
  • Du kan når som helst deaktivere stedstjenester i nettleserens innstillinger

2.3 Data vi IKKE samler inn

Tråkke samler IKKE inn:

  • Personnummer, fødselsdato eller andre sensitive personopplysninger
  • Kontaktinformasjon (e-post, telefonnummer, adresse)
  • Brukerkontoer eller påloggingsinformasjon
  • IP-adresser eller enhets-IDer for sporing
  • Analysedata om bruksmønstre
  • Informasjonskapsler (cookies) for sporing eller markedsføring

3. Hvordan brukes dataene?

Data som lagres lokalt brukes kun til:

  • Funksjonalitet: Sørge for at appen fungerer som forventet
  • Brukeropplevelse: Huske dine preferanser mellom økter
  • Offline-bruk: Tillate bruk av appen uten internettforbindelse
  • Ytelse: Cache kartfliser og data for raskere lasting

4. Deling med tredjeparter

Tråkke deler INGEN personopplysninger med tredjeparter for markedsføring, analyse eller sporing.

4.1 Eksterne datatjenester (GDPR-kompatible)

For å vise kartdata og stedsinformasjon bruker Tråkke følgende tjenester:

Tjeneste Leverandør Formål Personvern
Kartfliser Kartverket (Norge) Vise topografiske kart Norsk offentlig etat, GDPR-kompatibel
Satellittbilder Esri (USA) Satellittbilder / flyfoto ⚠️ Amerikansk selskap - IP-adresse sendes til USA
Stedssøk Nominatim / OpenStreetMap Søk etter steder og adresser Åpen kildekode, EU-basert
POI-data (turstier, hytter) Overpass API / OpenStreetMap Interessepunkter og turdata Åpen kildekode, EU-basert
DNT-hytter Den Norske Turistforening Informasjon om DNT-hytter Norsk medlemsorganisasjon
Tilfluktsrom DSB via Geonorge Offentlige tilfluktsrom Norsk offentlig etat, GDPR-kompatibel

Viktig: Når du laster kartdata fra disse tjenestene, kan de se din IP-adresse i sine server-logger. Dette er teknisk nødvendig for å levere tjenesten.

⚠️ Om Esri satellittbilder (USA)

Hvorfor bruker vi en amerikansk tjeneste?

Vi har grundig undersøkt alle norske og europeiske alternativer for satellittbilder:

  • Norge i bilder (Kartverket): Krever autentisering eller er utilgjengelig for offentlig bruk
  • Sentinel-2 (Kartverket/EU): Veldig treg lasting (dårlig brukeropplevelse)

Hva betyr dette for ditt personvern?

  • Når du velger satellittkart, lastes kartfliser fra Esri sine servere i USA (arcgisonline.com)
  • Din IP-adresse sendes til Esri for å hente kartfliser
  • Esri kan se hvilke kartområder du ser på (basert på hvilke fliser som lastes)
  • Data behandles i henhold til Esris personvernregler (underlagt amerikanske lover)

Hva gjør vi for å beskytte deg?

  • Vi sender INGEN bruker-ID, cookies eller sporingsdata til Esri
  • Du kan når som helst bytte til topografisk kart (100% norsk)
  • Satellittkart er valgfritt - du velger selv om du vil bruke det

Alternativ: Bruk kun topografisk kart (Kartverket) hvis du ønsker å unngå amerikanske tjenester helt.

4.2 Ingen amerikanske sporingstjenester

Tråkke bruker IKKE:

  • Google Analytics eller andre analysetjenester
  • Facebook Pixel eller andre markedsføringsverktøy
  • Google Fonts fra CDN (fonter er selvhostet)
  • Amerikanske cloud-tjenester for lagring av brukerdata

5. Dine rettigheter etter personvernforordningen (GDPR)

I henhold til personvernforordningen (GDPR) og personopplysningsloven har du følgende rettigheter:

5.1 Rett til innsyn (Artikkel 15)

Du har rett til å få informasjon om hvilke personopplysninger vi behandler om deg. Siden alle data lagres lokalt på din enhet, kan du se dette i nettleserens utviklerverktøy (DevTools → Application → Local Storage).

5.2 Rett til sletting (Artikkel 17 - "Retten til å bli glemt")

Du kan når som helst slette alle dine data fra Tråkke ved å:

  • Bruke knappen "Slett mine data" i appens innstillinger
  • Slette nettleserens lokale lagring manuelt
  • Avinstallere PWA-appen fra din enhet

5.3 Rett til dataportabilitet (Artikkel 20)

Siden alle data lagres lokalt, har du full kontroll og kan eksportere dataene ved å bruke nettleserens utviklerverktøy.

5.4 Rett til å protestere (Artikkel 21)

Du kan når som helst slutte å bruke Tråkke og slette alle lokale data.

5.5 Klagerett til tilsynsmyndighet

Du har rett til å klage til Datatilsynet hvis du mener vi behandler dine personopplysninger i strid med personvernregelverket:

  • Datatilsynet: www.datatilsynet.no
  • E-post: postkasse@datatilsynet.no
  • Telefon: 22 39 69 00

6. Datasikkerhet

Vi tar datasikkerhet på alvor og har implementert følgende tiltak:

  • HTTPS: All kommunikasjon med appen er kryptert via HTTPS
  • Lokal lagring: Data lagres kun på din enhet, ikke på eksterne servere
  • Service Worker sikkerhet: PWA-funksjonalitet følger moderne sikkerhetsstandarder
  • Ingen tredjeparts-trackere: Vi bruker ingen sporingsverktøy eller analyse-cookies
  • Åpen kildekode: Koden er offentlig tilgjengelig på GitHub for gjennomsiktighet

7. Informasjonskapsler (Cookies)

Tråkke bruker IKKE informasjonskapsler (cookies) for sporing, analyse eller markedsføring.

Nettleseren kan opprette tekniske cookies for å håndtere:

  • Service Worker-funksjonalitet (nødvendig for PWA)
  • Sesjonshåndtering (teknisk nødvendig)

Disse er strengt nødvendige for at appen skal fungere og krever ikke samtykke i henhold til GDPR.

8. Barn og personvern

Tråkke samler ikke inn personopplysninger fra brukere under 18 år. Siden vi ikke krever registrering eller lagrer personlige data, kan appen brukes trygt av alle aldersgrupper under voksen veiledning.

9. Endringer i personvernerklæringen

Vi kan oppdatere denne personvernerklæringen fra tid til annen. Vesentlige endringer vil bli kommunisert via:

  • Oppdatert "Sist oppdatert"-dato øverst på denne siden
  • Informasjon i appen ved neste besøk (hvis relevant)
  • GitHub-repository for full endringshistorikk

10. Kontaktinformasjon

Hvis du har spørsmål om denne personvernerklæringen eller behandling av personopplysninger, kan du kontakte oss via:

  • GitHub Issues: github.com/elzacka/trakke-react/issues
  • GitHub Discussions: github.com/elzacka/trakke-react/discussions

11. Juridisk grunnlag

Behandling av personopplysninger i Tråkke baseres på:

  • Artikkel 6(1)(f) - Berettiget interesse: Lagring av preferanser og cache-data er nødvendig for å levere tjenesten
  • Artikkel 6(1)(a) - Samtykke: Bruk av stedstjenester krever eksplisitt samtykke fra bruker via nettleserens tillatelsesdialog

📍 Slett mine data

For å slette alle dine lokalt lagrede data fra Tråkke, gå til appens innstillinger og velg "Slett mine data". Dette vil fjerne all informasjon lagret på din enhet, inkludert:

  • Kartvisningsinnstillinger
  • Søkehistorikk
  • Brukerpreferanser
  • Cache-data (offline-kart)

Alternativt kan du slette nettleserens lokale lagring manuelt i nettleserens innstillinger.
